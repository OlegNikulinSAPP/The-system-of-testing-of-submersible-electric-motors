# Система тестирования погружных электродвигателей

## Описание
Программное обеспечение для тестирования погружных электродвигателей (REDA). 
Приложение предоставляет интерфейс для проведения различных видов испытаний 
и анализа результатов.

## Функциональность
- Проведение 15 различных видов испытаний
- Работа с базой данных результатов
- Формирование отчетов
- Визуализация данных

## Требования
- **Qt 5.x** или выше (с модулями: core, gui, sql, serialbus, serialport)
- **Компилятор** с поддержкой C++11
- **Windows** (приложение использует RC-файлы для иконки)

## Сборка проекта
1. Установите Qt с компонентами:
   - Qt Creator
   - Компилятор (MinGW или MSVC)
   - Модули: Qt SerialBus, Qt SerialPort
2. Откройте файл `QtProjectRedaNew_version_2.pro` в Qt Creator
3. Настройте комплект сборки (Kit)
4. Запустите сборку (Ctrl+B)

## Структура проекта
- `main.cpp` - точка входа
- `mainwindow.*` - главное окно
- `password.*` - окно авторизации
- `window_experience_1-15.*` - окна для 15 видов испытаний
- `Resource.qrc` - ресурсы приложения
- `logo.rc` - файл ресурсов для иконки
